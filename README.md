# Installation
### Download and change folder
```
cd ~
git clone https://github.com/sbennell/linuxsetup.git

cd linuxsetup
```
### Run install
```
chmod +x ./install.sh
./install.sh
```

# Uninstallation

### Change to dotfiles folder
```
cd ~/linuxsetup
```

### Run uninstaller
```
./uninstall.sh
```
